# VISUAL SORT — ソートアルゴリズムビジュアライザー

ブラウザ上で動く、ソートアルゴリズムの可視化ツールです。古い蛍光体モニター／オシロスコープ風の画面に、棒の高さで配列を表示し、比較・交換・書き込みの様子をリアルタイムに描きます。依存ライブラリなし・単一 HTML ファイルで動作します。

Claude Codeに丸投げして作ってもらいました。thanks claude code.

## デモ

次のリンクをクリックすると、GitHub Pages上でそのまま動作します。

[・Visual Sort demoタイトル](https://uchiko.github.io/visualsort/)　

## 機能

- **7 種類のアルゴリズム** — バブル / 選択 / 挿入 / クイック / マージ / ヒープ / シェル
- **同じ配列のまま整列法を切り替え** — アルゴリズムごとの手数や挙動を見比べられる
- **状態の色分け** — 未整列 / 比較 / 書き込み / ピボット / 確定
- **要素数（8〜160）と速度（約 2〜4000 操作/秒、対数）** をスライダーで調整
- **比較回数・書き込み回数**をリアルタイム表示。完了時は左から右への確定スイープで締める
- **音（任意・初期はオフ）** — 値を音程に変換して鳴らす
- 一時停止／再開・シャッフル対応、モバイル幅にも追従

## 仕組み

各アルゴリズムは配列を直接アニメーションさせるのではなく、まず

- `cmp`（比較）
- `swap`（交換）
- `set`（書き込み）
- `pivot`（目印）
- `sorted`（確定）

という **操作列（ops）** を記録する純粋関数として実装しています。記録した操作列を `requestAnimationFrame` で再生する方式のため、

- 一時停止・再開が素直に書ける
- 速度を可変にしても、1 フレームで処理する操作数を増減するだけで済む
- マージソートのように「交換」ではなく「上書き」で進む手法も同じ枠組みで扱える

という利点があります。

## 使い方

1. アルゴリズムのチップを選ぶ（配列は維持されます）
2. 要素数・速度を調整
3. **▶ 実行** で再生、**⏸ 一時停止** で停止
4. **↻ シャッフル** で新しい配列を生成

## ローカルで開く

ファイルを直接開くだけでも動きますが、簡易サーバ経由でも確認できます。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
